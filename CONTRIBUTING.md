# Contributing to Error Translator

Welcome! Your contribution has been received and will be translated into emotional human language shortly.

> **Therapist persona:** "It's really brave of you to open this CONTRIBUTING.md. That tells me you're ready to grow."

## Getting Started

```bash
git clone https://github.com/arnoldwender/error-translator.git
cd error-translator
npm install
npm run dev
```

The app should be running at `http://localhost:5173`. Start translating your feelings — err, errors.

## How to Contribute

1. **Fork the repo** and create a feature branch (`feat/new-error-translations`)
2. **Make your changes** — new errors, new personas, new languages
3. **Test locally** — make sure the translations are actually funny
4. **Submit a PR** with a clear description

## Code Style

- TypeScript, strictly typed — `undefined is not a function` hits different when it's your own code
- React functional components with hooks
- Tailwind CSS for styling
- Each persona must have a distinct, consistent voice
- Translations should be funny AND somewhat educational — bonus points if someone learns what the error actually means

## Ideas We Love

- New translator personas (the overenthusiastic junior dev, the zen monk, the conspiracy theorist)
- More errors from more languages (Ruby, PHP, Swift, Kotlin)
- Persona-specific reactions to the same error — the contrast is where the comedy lives
- Interactive features (error severity voting, community favorites)

## Bug Reports

Open an issue with the error message and what went wrong. If an error translation accidentally makes someone feel BETTER about their bug, that's working as intended.

## Feature Requests

Know a cryptic error message that deserves the emotional treatment? Open an issue with the `enhancement` label. Include the raw error and, if you're feeling creative, a sample translation.

## The Golden Rule

Error messages are a universal developer pain point. The humor should come from recognition — that moment of "oh god, I've seen that error at 3am" — not from making anyone feel dumb for encountering it.
